Atlas Studio

Este archivo funciona sin conexión. El proyecto guardado conserva toda la información histórica, visual y editorial.

Flujo de publicación: guarda reli-tree-project.json, súbelo a data/reli-tree-project.json en el repositorio y GitHub Pages reconstruirá la web.

Herramientas: selecciona y arrastra nodos; usa el recuadro de selección múltiple para editar o eliminar grupos; crea uniones pulsando primero el origen y luego el destino; añade acontecimientos o transforma visualmente la referencia.

Relaciones: sus extremos permanecen unidos a las entidades. Puedes elegir un recorrido recto, ortogonal o curvo, añadir puntos de paso por otros territorios y asignar un color a cada etapa para crear degradados geográficos.

Seguridad: toda eliminación —entidad, relación, acontecimiento, área, punto de paso, marca temporal, icono o referencia— exige confirmación.

Referencia: puedes cargar cualquier imagen como capa semitransparente, moverla, girarla y redimensionarla. Mayús conserva su proporción y Alt transforma desde el centro. No se incorpora al archivo guardado salvo que marques expresamente «Incrustar referencia».

Tableros: el fondo global, la cuadrícula y cada área tienen color, opacidad, contorno, forma y cabecera independientes. Los preajustes permiten pasar de columnas suaves a contornos, zonas sólidas o un tablero sin bandas.
