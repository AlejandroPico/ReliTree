# Iconos SVG de entidades

Coloca aquí los iconos vectoriales del atlas.

ReliTree intenta asociarlos automáticamente por este orden:

1. identificador de la entidad, por ejemplo `christianity.svg`;
2. nombre normalizado, por ejemplo `cristianismo.svg`.

También se puede indicar una ruta concreta o incrustar un SVG desde el editor. Los iconos añadidos a esta carpeta deben tener licencia compatible con la publicación pública del repositorio.
