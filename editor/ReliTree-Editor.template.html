ReliTree Editor

Este archivo funciona sin conexión. El proyecto guardado conserva toda la información histórica, visual y editorial.

Flujo de publicación: guarda reli-tree-project.json, súbelo a data/reli-tree-project.json en el repositorio y GitHub Pages reconstruirá la web.

Herramientas: selecciona y arrastra nodos para cambiar territorio/año; usa la selección múltiple para editar o eliminar grupos; crea uniones pulsando primero el origen y luego el destino; añade acontecimientos regionales o vinculados a entidades concretas.

Relaciones: sus extremos permanecen unidos a las entidades. Puedes elegir un recorrido recto, ortogonal o curvo, añadir puntos de paso por otros territorios y asignar un color a cada etapa para crear degradados geográficos.

Seguridad: toda eliminación —entidad, relación, acontecimiento, área, punto de paso, marca temporal, icono o referencia— exige confirmación.

Referencia: puedes cargar el cartel original como capa semitransparente. No se incorpora al archivo guardado salvo que marques expresamente «Incrustar referencia».
